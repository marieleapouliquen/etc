Modèle 1: l'équilibre radiatif¶

Présentation¶

Considrons une plante du systme solaire (Terre, Mars ou Vnus). Cette plante reoit une certaine quantit d'nergie solaire, fonction de sa distance au Soleil. Cette nergie dite incidente est intercepte par l'atmosphre plantaire. Les composs atmosphriques tels que les nuages et les gaz rflchissent une partie de l'nergie incidente, qui est donc renvoye vers l'espace. La fraction d'nergie solaire rflchie sur l'nergie solaire reue est appele albedo. Ce paramtre varie selon la distance de la plante au Soleil et la composition de l'atmosphre plantaire.

L'nergie restante est transforme en chaleur par les composs atmosphriques et la surface terrestre. La chaleur ainsi produite par la plante est rayonne dans toutes les directions, y compris vers l'espace. Lorsque la plante est en quilibre, c'est--dire lorsque sa temprature est constante dans le temps, l'nergie solaire incidente est gale l'nergie sortante, qui est la somme de l'nergie solaire rflchie et de la chaleur mise par la plante. Ce modle se propose de calculer la temprature d'quilibre radiatif d'une plante, en fonction de la quantit d'nergie solaire qu'elle reoit.

import numpy
import matplotlib.pyplot as plt

def nakedplanet(alpha=0.3,L=1350):
    # PARAMETRES
    timeStep = 100          # years
    waterDepth = 4000       # meters
    Cp = waterDepth * 4.2E6 # J/K m2
    epsilon = 1
    sigma = 5.67E-8         # W/m2 K4
    Teq = ( L * (1 - alpha) / (4 * epsilon * sigma) ) ** (1/4)

    # INITIALISATION
    time = []
    temperature = []
    heatContent = []

    time.append(0)
    temperature.append(0)
    heatContent.append(Cp * temperature[0]) # J/m2
    heatIn = L * (1 - alpha) / 4
    heatOut = 0

    # INTEGRATION
    for it in range(1,100):
        time.append(time[it-1] + timeStep)
        heatOut = epsilon * sigma * temperature[it-1]**4
        heatContent.append( heatContent[it-1] + (heatIn - heatOut) * timeStep * 3.14E7)
        temperature.append( heatContent[it] / Cp )

    return temperature, time
